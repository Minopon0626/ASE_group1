
sudo systemctl start pigpiod
    上記のコマンドを実行してからmain.pyを動かすように！！！！

main.pyはすべてを連動させて動かすプログラム。現状は以下の機能が実装されている。
このプログラムのあるディレクトリにlogというディレクトリがあるか確認する。なければ作成する。
以下を繰り返す
    撮影する
        撮影したものはmain.pyと同じディレクトリに保存される。名前はcaptured_image.jpgで保存される。
    現在時刻を取得する
    logディレクトリの中に現在時刻の名前のディレクトリを生成する
    作成したディレクトリの中に撮影した画像をコピーする
    captured_image.jpgの名前を参照して画像分析を行う
        画像分析を行い認知した物体は、物体ごとにその範囲をトリミングして作成したディレクトリに保存する
    画像分析を行った結果人が一人でも写っていれば赤外線出力を行う

create_or_find_output.pyは名前とパスを受け取り指定したパスのディレクトリに名前と一致する子ディレクトリがない場合ディレクトリを作成する。そしてそのディレクトリの完全パスを返す
file_path_finder.pyは名前を受け取り現在のディレクトリおよびサブディレクトリ内で指定されたファイルを検索し、なまえが一致するディレクトリが存在する場合はその完全パスを返す。
ImageDetection.pyは画像のパスと出力先のディレクトリを受け取りその画像内の物体を検出してトリミングした画像を出力先のディレクトリに入れるプログラム
Infrared_rays_send.pyはcgirを利用して赤外線信号を送信するプログラム
old_create_or_find_output.pyは指定されたディレクトリが存在しない場合は作成し、存在する場合はその完全パスを返す。これは新規で作成した際に保存しているだけでmain.pyでは使用していない
photographing.pyは関数が読みだされた際に一回撮影するpythonプログラム。撮影して保存したファイルパスを返す。この際保存するディレクトリはこのプログラムを動かしたディレクトリである
time_capture.pyは現在の日時を取得し、フォーマットする関数

main.pyは既存の出力方法を試すプログラム
new_main.pyは新しい出力方法を試したもの　ただし各プログラムにprint文が多く含まれているためおそらくうまくいかない
